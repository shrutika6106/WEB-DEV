# Web Development Practicals

This repository contains simple web development practicals created using **HTML and JavaScript**. These programs are designed for **college lab work and basic practice**.

## Practicals Included

### 1. Simple Calculator (`index.html`)

A basic calculator web page that accepts four numbers from the user and performs **addition**.

**Functionality:**

* Takes four numeric inputs
* Adds all the numbers
* Displays the result on the webpage

**Concepts used:**

* HTML input elements
* JavaScript function
* `parseInt()` for number conversion

---

### 2. Text to Speech (`speech.html`)

A web page that converts entered text into spoken voice using browser text-to-speech features.

**Concepts used:**

* JavaScript Speech Synthesis
* User input handling

---

### 3. To-Do List Webpage (`todo.html`)

A simple To-Do List application that allows users to add and view tasks.


---

## How to Run

1. Download or clone the repository
2. Open any `.html` file in a web browser
3. No additional software or installation is required

## Purpose

These practicals are created for **educational purposes** and to understand the basics of web development.
